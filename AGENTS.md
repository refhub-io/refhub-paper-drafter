# refhub-paper-drafter

This plugin provides the `refhub-paper-drafter` skill for drafting HCI and visualization research papers.

The skill takes scattered notes (local files + RefHub vault queries) and produces a grounded manuscript draft modeled on best practices from award-winning HCI and visualization research.

**Invoke it** by asking the agent to draft a paper, or by typing the skill name in your harness.

**Requires:** `refhub-cli` — see https://github.com/refhub-io/refhub-cli. For Codex: https://github.com/refhub-io/refhub-codex.
